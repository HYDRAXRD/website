

# HydraXRD — Memecoin Battle Game Website

## Overview
A single-page, modern, vibrant landing site for HydraXRD — a memecoin battle game on the Radix blockchain. The uploaded Hydra logo will be featured prominently throughout the site.

## Design Direction
- **Dark theme** base (deep navy/black) with Radix blue as primary, accented by purple, neon green, and orange
- Bold, energetic typography with glowing/neon effects
- Smooth scroll animations and hover effects (glows, scale, parallax)
- Fully mobile-responsive
- Meme-culture energy balanced with crypto-project professionalism

## Sections (top to bottom)

### 1. Navigation Bar
- HydraXRD logo + name (sticky top)
- Links: About, Game, Roadmap, Tokenomics, Community
- "Connect Wallet" button (visual only — no actual wallet integration)

### 2. Hero Section
- Large Hydra logo with glow/pulse animation
- Headline: "HydraXRD: Unite the Radix Meme Revolution"
- Subheadline about the first memecoin battle game
- Two CTA buttons: "Connect Wallet" and "View Roadmap" (scrolls to roadmap)
- Animated particle/network background effect using CSS

### 3. About Section — "Integrating, Not Dividing"
- Mission statement text as provided
- Icon cards highlighting key values (Unity, Community, Growth)

### 4. Game Concept — "Battle Within the Radix Narrative"
- Feature cards: Turn-based battles, meme integration, character collection, strategic combat, community content
- Visual battle arena mockup area with placeholder illustrations
- Engaging layout with icons and animations

### 5. Roadmap — "The Path Forward"
- Vertical timeline with all 5 phases (Q1 2026 through 2027)
- Each phase as an expandable or visually distinct card with all bullet points
- Color-coded phase indicators

### 6. Tokenomics
- Token utility list with icons (in-game currency, battle fees, upgrades, staking, governance)
- Visual distribution chart (placeholder pie/donut chart using Recharts)
- LP lock badge/callout

### 7. Community Section
- "Join the Revolution" headline
- Social media buttons: Twitter, Telegram, Discord (links as placeholder #)
- Placeholder community stats (members, battles, memes created)

### 8. Footer
- Radix ecosystem links
- Legal disclaimer text
- Contact info
- Copyright

## Technical Approach
- Single-page app with smooth scroll navigation
- All content is static (no backend needed)
- CSS animations for particles, glows, and hover effects
- Recharts for tokenomics chart
- Lucide icons throughout
- The uploaded Hydra logo embedded as the main brand asset
